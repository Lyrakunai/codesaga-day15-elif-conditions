## 🧠 What I Learned
- `elif` means **else if**.
- `elif` runs only when the previous condition is False.
- Python checks conditions from top to bottom.
- In one `if-elif-else` chain, only the first True block executes.
- A new `if` starts a new decision chain.
- Condition order matters when conditions overlap.
- Dry Run helps understand program execution before running code.
